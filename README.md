# Sales Performance Dashboard (Power BI)

## Overview

This project presents an interactive Sales Performance Dashboard built using Power BI. The dashboard analyzes sales data from the Superstore dataset and provides key business insights through visualizations and KPIs.

## Dataset

* **Dataset:** Sample Superstore
* **Format:** CSV
* **Tool Used:** Microsoft Power BI

## Dashboard Features

* Total Sales
* Total Profit
* Total Orders
* Sales by Category
* Sales by Region
* Monthly Sales Trend
* Top Performing Products
* Interactive Filters (Region, Category, Segment)

## Key Insights

* Identifies the highest-selling product categories.
* Compares sales performance across different regions.
* Shows monthly sales trends.
* Highlights top-performing products based on sales.
* Provides an overview of overall business performance.

## Tools & Technologies

* Microsoft Power BI
* Power Query
* DAX
* CSV Dataset

## Files

* `Sample - Superstore.csv` – Dataset
* `Sales Dashboard.pbix` – Power BI project file
* `README.md` – Project documentation

## How to Use

1. Open the `.pbix` file in Microsoft Power BI Desktop.
2. Refresh the dataset if required.
3. Explore the dashboard using the available filters and slicers.
4. Interact with charts to gain business insights.

This project demonstrates:
